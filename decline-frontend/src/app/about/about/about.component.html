About Us

Decline is a free-to-use web application designed to help users practice German adjective declensions. By generating random sentences, it provides an opportunity to practice various article and adjective forms. This project was created as a hobby.

Usage

Each question is accompanied by two columns from which you can select the correct article and adjective form. You can then check your answer using the "Check Answer" button. The application can be used without registration, but if you register, you can track your correct and incorrect answers in your profile and retry the questions you answered incorrectly.
